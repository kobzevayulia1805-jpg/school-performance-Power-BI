# Stage 4: Final Dashboard (HW12)

## Objective

Use built-in Power BI analytics tools, create quick measures for time analysis,
and unify all sheets into a single, stylistically consistent dashboard with
seamless navigation.

## What was done

- **4 sheets unified** into a single navigational dashboard:
  - Academic Performance Overview
  - Class Performance
  - Class Details
  - Analytics & Time Intelligence
- **Built-in analytics**:
  - Trend line on Average Grade by Date
  - Forecasting (visible gray area extending into future months)
  - P25 and P75 percentile lines for context
- **Quick measures** for time intelligence:
  - YTD comparison
  - MoM% change
- **Custom navigation menu** at the top — buttons that switch between sheets
- **Unified design**: consistent color palette (blue / white),
  typography, and layout across all pages
- **Two filter strategies**:
  - Page-level: Type of Grade, Academic Year, Department Hint
  - Cross-visual: automatic filtering when clicking on charts
